# bar-app
ChatGPTと作るモバイルオーダーアプリ

# dir構成について
bar-app/
│
├── src/
│   ├── components/  # UI コンポーネント
│   ├── pages/       # 各ページ (ログイン、メインメニュー、管理者ダッシュボード等)
│   ├── services/    # Firebaseなどの外部サービス連携
│   ├── hooks/       # カスタムフック
│   ├── styles/      # 共通スタイル
│   └── utils/       # ユーティリティ関数
├── public/          # 公開ファイル
└── firebase.json    # Firebaseの設定ファイル
